# Landing Page

Landing Page para promocionar un producto o servicio, información del producto relevante, facil de entender y destacable para aumentar la tasa de conversión.

## Historias de usuario

Utilice las siguientes historias de usuario para desarrollar esta aplicación

- El usuario puede `ver` en la página una o más imágenes del producto
- El usuario puede `ver` una lista con todas las características del producto
- El usuario puede `ver` cómo este producto mejorará la vida del comprador. ¿Por qué debería comprarlo?
- El usuario puede `ver` una sección de contacto (un texto con el correo electrónico)

## Caracteristicas adicionales:
- El usuario puede `ver` una sección de preguntas frecuentes.
- El usuario puede `ponerse` en contacto con los miembros del personal a través de un formulario de contacto.
- El usuario puede `registrarse` para recibir notificaciones sobre el producto.
- El usuario puede `comprar` el producto.

## Historias de usuario v2

Utilice las siguientes historias de usuario para desarrollar la mejora de esta aplicación

- El usuario puede hacer clic en un botón `Ver productos` de la página de inicio para visualizar la página de productos.
- El usuario puede ver una tarjeta en la página de productos para cada producto, que muestra la miniatura del producto, el nombre, el precio, una breve descripción y un botón `Seleccionar`.
- El usuario puede ver una página de detalles del producto que se muestra cuando se hace clic en el botón `Seleccionar`, que muestra la misma información que la tarjeta del producto, pero también un ID de producto único, una descripción larga, un botón `Agregar al Carrito` y un botón `Ver más productos`.
- El usuario puede ver un mensaje de confirmación cuando el producto se agrega al carrito de compras.
- El usuario puede hacer clic en `Ver más productos` para volver a la página de productos.
- El usuario puede hacer clic en un botón `Carrito de compras` tanto en la página de inicio como en la página de productos.
- El usuario puede hacer clic en un botón `Carrito de compras` para mostrar la página del carrito de compras, que contiene los campos para introducir el ID del producto, el nombre, el precio y la cantidad solicitada para cada producto añadido previamente al carrito.
- El usuario puede ver el importe total de la compra en el carrito, que se calcula como la suma de las cantidades multiplicadas por el precio unitario de cada producto pedido.
- El usuario puede ajustar la cantidad solicitada de cualquier producto para modificar el importe total de la compra.
- El usuario puede hacer clic en un botón `Confirmar orden de compra` en la página del carrito de compras para completar el pedido. Verá un número de confirmación una vez que se haya realizado el pedido.
- El usuario puede hacer clic en un botón `Cancelar orden de compra` en la página del carrito de compras para cancelar el pedido. Verá que las cantidades de los productos y el importe total de la compra se restablecen a cero.
- El usuario puede hacer clic en un botón `Ver más productos` en la página del carrito de compras para regresar a la página de productos. Si el pedido aún no se ha realizado, esto no eliminará los productos que ya se hayan agregado a la página de productos.

## Caracteristicas adicionales:
- El usuario podrá ver un mensaje de error si la cantidad solicitada supera la cantidad disponible del producto.
- El usuario puede especificar una dirección de facturación y una dirección de envío al realizar el pedido desde la página del carrito de compras.
- El usuario puede ver los gastos de envío añadidos al importe total de la compra.
- El usuario puede ver los impuestos sobre las ventas añadidos al importe total de la compra.
- El desarrollador implementará el inventario de productos en un archivo externo o en una base de datos.
